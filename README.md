# Calculadora em Shell Script e Python

Este repositório contém dois projetos simples desenvolvidos como parte do curso de Linux e Python:

1. **Um script executável em Shell (`calculadora.sh`)**
2. **Uma calculadora em Python (`calculadora.py`, caso você adicione)**  
3. **Um arquivo com os comandos utilizados durante a atividade (`comandos.txt`)**

---

##  ** Arquivos do repositório**

| Arquivo | Descrição |
|--------|-----------|
| `calculadora.sh` | Calculadora desenvolvida em Shell Script. |
| `comandos.txt` | Comandos utilizados na criação, permissão e execução do script. |
| `README.md` | Documentação do projeto. |
| *(Opcional)* `calculadora.py` | Versão da calculadora em Python, se desejar incluir. |
